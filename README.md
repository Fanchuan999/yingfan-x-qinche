# 应帆×秦彻 · 专属回忆

## 如何启动

双击 **启动.bat** 即可在浏览器中打开网页。

或者直接用浏览器打开 `index.html`。

---

## 🎨 功能说明

- **西方龙剪影**：首页背景有浮动龙影动画
- **乌鸦元素**：分布在页面两侧
- **火焰粒子**：全屏飘浮的余烬粒子特效
- **照片墙**：点击可放大查看，支持键盘左右箭头切换，**支持上传自己的照片**
- **语录轮播**：每 5 秒自动切换，也可点击圆点手动切换，**支持添加自己的语录**
- **键盘快捷键**：照片放大后 ESC 关闭，← → 切换
- **音乐播放器**：底部迷你播放器，支持播放/暂停、上下切歌、列表循环/单曲循环、拖拽排序播放列表、点击进度条跳转
- **内容管理**：导航栏「更多」→「内容管理」可管理上传的照片和语录

---

## 📷 如何上传照片

直接在网页上操作，不需要修改代码：

1. 滚动到「照片墙」区域
2. 点击「＋ 上传照片」按钮
3. 选择你的照片（支持 JPG/PNG/GIF，单张不超过 20MB）
4. 照片自动出现在照片墙中
5. 悬停在照片上可看到删除按钮，点击即可删除

> 照片存在你浏览器的本地存储中，换电脑/清缓存会丢失

---

## 💬 如何添加语录

1. 滚动到「语录」区域
2. 点击「＋ 添加语录」按钮
3. 输入你想说的话，点击「添加」
4. 语录混入轮播中（金色圆点标识）

---

## 🗂️ 内容管理

点击导航栏「更多」→「内容管理」：

- **照片管理**：查看/删除已上传照片，支持拖拽上传
- **语录管理**：查看/删除自定义语录
- **重置数据**：清除所有个人数据，恢复默认

---

## 🎵 如何添加/更换音乐

1. 把 MP3 文件复制到 `music` 文件夹
2. 用记事本打开 `index.html`
3. 找到 `const musicFiles = [` 这一行
4. 在数组里添加或修改文件名即可
5. 支持拖拽排序播放列表（点击播放器右侧 ☰ 按钮打开列表）
